# quick-notes

通用速记技能 - 支持多分类的笔记记录与同步

## 功能

- ✅ 多分类支持（育儿、工作、读书等）
- ✅ 多种日期格式支持
- ✅ 智能去重
- ✅ 多数据源合并
- ✅ 自动同步坚果云
- ✅ **编辑/删除笔记**（新增）
- ✅ **全文搜索**（新增）
- ✅ **健康检查**（新增）
- ✅ **本地备份**（新增）
- ✅ **动态分类管理**（新增）
- ✅ **多格式导出**（新增）

## 触发关键词

- `#笔记`
- `#育儿笔记`
- `速记`

## 使用方式

### 添加笔记（指定分类）
```
#笔记 育儿 宝宝今天会走路了
#笔记 工作 完成了需求评审
#笔记 读书 读完了《百年孤独》
```

### 添加笔记（指定日期）
```
#笔记 育儿 2026-03-16,宝宝会走路了
#笔记 3月16日,宝宝会走路了
```

### 查询笔记
```
#笔记 育儿 2026-03-16    # 查看指定日期（显示 ID 便于编辑/删除）
#笔记 育儿              # 查看最近7天
#笔记 分类              # 查看所有分类
```

### 编辑/删除笔记
```
#笔记 编辑 ID:2026-03-19_abc123 修改后的内容
#笔记 删除 ID:2026-03-19_abc123
```
> 查询笔记时会显示 ID，格式为 `日期_内容哈希`，便于识别

### 搜索笔记
```
#笔记 搜索 关键词        # 全分类搜索
#笔记 育儿 搜索 关键词   # 指定分类搜索
```

### 健康检查
```
#笔记 健康检查           # 检测凭证有效性和云端连通性
```

### 分类管理
```
#笔记 添加分类 生活      # 添加新分类
#笔记 添加分类 生活 日常  # 添加分类并指定显示名称
#笔记 删除分类 生活      # 删除分类（数据文件保留）
#笔记 重命名 生活 日常   # 重命名分类
```

### 导出笔记
```
#笔记 导出 markdown      # 导出为 Markdown
#笔记 导出 json          # 导出为 JSON
#笔记 导出 html          # 导出为 HTML（带样式）
```
> 导出文件保存在 skill 目录下的 `exports/` 文件夹

### 维护命令
```
#笔记 去重      # 清理重复记录
#笔记 合并      # 合并多数据源
#笔记 同步      # 手动同步到云端
```

## 凭证

需要以下环境变量：

| 变量 | 说明 |
|------|------|
| `JIANGUOYUN_EMAIL` | 坚果云账号邮箱 (必填) |
| `JIANGUOYUN_PASSWORD` | 坚果云应用密码 (必填) |

## 日期格式支持

| 格式 | 示例 |
|------|------|
| 标准 | 2026-03-16 |
| 中文 | 3月16日 |
| 缩写 | 3-16 |
| 自然 | 今天、昨天 |

## 笔记 ID 说明

笔记 ID 格式为 `日期_内容哈希`，例如 `2026-03-19_abc123`：
- 便于识别是哪天的笔记
- 同一天多条笔记不冲突
- 查询时直接显示，方便复制使用

## 本地备份

每次写入前自动备份到 `.backup/` 目录，保留最近 5 份：
- 备份文件带时间戳
- 自动清理旧备份
- 可在配置中关闭或调整备份数量

## 配置示例

```json
{
  "categories": {
    "default": { "name": "默认笔记", "file": "notes.csv", "cloud_path": "/apps/notes/" },
    "育儿": { "name": "育儿笔记", "file": "parenting.csv", "cloud_path": "/apps/parenting/" }
  },
  "backup": {
    "enabled": true,
    "max_count": 5,
    "dir": ".backup"
  }
}
```
